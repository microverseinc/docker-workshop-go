## 🛫 Getting Started

### Prerequisites

1. Verify if Go is installed

```sh
go version
```

### Installing

1. Clone the project to your machine:

```sh
git clone https://github.com/microverseinc/docker-workshop-go
```

2. Install dependencies

```sh
go get .
```

### Executing program

1. Run the application

```sh
go run .
```
